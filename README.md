# StringListManager

A simple and interactive Java program that demonstrates how to use the `ArrayList` class to store, manipulate, and manage a dynamic collection of strings. 

This project is designed to help understand core Java concepts such as dynamic sizing, collection manipulation, and traversing lists using the enhanced `for-each` loop.

---

## 🚀 Features

* **Add Elements:** Dynamically insert new strings into the list.
* **Remove Elements:** Delete existing items from the collection.
* **Search Elements:** Quickly look up whether a specific string exists in the list.
* **Display List:** Print all elements sequentially using an efficient for-each loop.

---

## 🛠️ Concepts Covered

* **`java.util.ArrayList`**: Utilizing dynamic arrays that grow and shrink automatically.
* **Collection Methods**: Practicing built-in methods like `.add()`, `.remove()`, and `.contains()`.
* **Enhanced For-Loop**: Clean and readable iteration over collection elements.
